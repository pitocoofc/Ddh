API de Scripts (JavaScript real)

Cada script roda a cada frame enquanto o objeto existir no modo TESTE. Variáveis disponíveis dentro do código:

self — o próprio objeto (self.position.x/y/z, self.rotation.y, self.scale, self.name)
dt — segundos desde o último frame
keys — teclado, ex: keys['w']
joystick — {x, y} do analógico virtual (-1 a 1)
vars — objeto global compartilhado com o HUD, ex: vars.pontos = 10
getObject(nome) — retorna outro objeto pelo nome
player — o objeto do jogador (câmera)
THREE — biblioteca Three.js inteira, para criar geometria, vetores, etc.

Exemplo — girar um objeto:
self.rotation.y += dt;

Exemplo — somar pontos ao encostar:
if(self.position.distanceTo(player.position) < 1.5) vars.pontos = (vars.pontos||0) + 1;

Script Geral roda uma única vez ao clicar em INICIAR TESTE — bom para inicializar variáveis.

Botões de GUI executam seu próprio script quando clicados, com acesso a vars, getObject e player.
